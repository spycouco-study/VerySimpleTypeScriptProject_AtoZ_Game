## 게임 프로토타입 사양서 (뱀 게임)

### 1. 게임 화면 설정
*   **화면 크기:** 가로 20칸, 세로 20칸의 격자형 화면으로 구성됩니다. 각 칸은 뱀이나 먹이의 한 부분을 나타냅니다.
*   **칸(블록) 크기:** 각 칸은 25x25 픽셀 크기로 표시됩니다.
*   **배경 색상:** 게임 화면의 배경은 어두운 회색으로 설정됩니다.

### 2. 뱀의 움직임 및 상호작용
*   **뱀의 외형:** 뱀의 머리 부분은 빨간색으로, 몸통 부분은 초록색으로 표시됩니다.
*   **벽 충돌:** 뱀이 게임 화면의 가장자리에 부딪히면 게임이 종료됩니다.
*   **초기 길이:** 뱀은 게임 시작 시 몸통 3칸의 길이로 시작합니다.
*   **이동 속도:** 뱀은 보통 빠르기(예: 약 150ms 간격으로 한 칸씩 이동)로 움직입니다.
*   **자기 몸통 충돌:** 뱀이 자신의 몸통에 부딪히면 게임이 종료됩니다.
*   **180도 방향 전환 제한:** 뱀이 현재 이동하는 방향의 정반대 방향으로는 즉시 방향을 전환할 수 없습니다 (예: 오른쪽으로 이동 중일 때 즉시 왼쪽으로 방향 전환 불가).
*   **조작 방법:** 키보드의 화살표 키(↑, ↓, ←, →)를 사용하여 뱀의 이동 방향을 조종합니다.

### 3. 먹이 시스템
*   **먹이 외형:** 먹이는 빨간색 사각형 모양으로 표시됩니다.
*   **먹이 생성:** 한 번에 1개의 먹이가 게임 화면 내 무작위 위치에 나타납니다. 먹이는 뱀의 몸통이나 다른 먹이와 겹치지 않는 곳에 생성되어야 합니다.
*   **먹이 섭취 효과:** 뱀이 먹이를 먹으면 뱀의 길이가 1칸 길어지고, 점수가 증가합니다.

### 4. 점수 시스템
*   **점수 계산:** 뱀이 먹이를 먹을 때마다 10점씩 점수가 증가합니다.
*   **점수 표시:** 현재 점수는 게임 화면의 왼쪽 위에 표시됩니다.

### 5. 게임 시작, 종료 및 재시작
*   **게임 시작 흐름:** 웹 페이지 로딩 후 '게임을 시작하려면 엔터 키를 누르세요' 메시지가 화면 중앙에 표시되며, 사용자가 Enter 키를 누르면 게임이 시작됩니다.
*   **게임 종료 조건:** 뱀이 벽에 부딪히거나 자신의 몸통에 부딪히면 게임이 종료됩니다.
*   **게임 종료 표시:** 게임 종료 시 화면 중앙에 '게임 오버' 메시지가 표시됩니다.
*   **재시작:** 게임 종료 후, 사용자는 'R' 키를 눌러 게임을 다시 시작할 수 있습니다.
*   **일시 정지 기능:** 게임 플레이 중 'P' 키를 누르면 게임이 일시 정지되며, 다시 'P' 키를 누르면 게임이 이어서 진행됩니다. 일시 정지 상태에서는 화면 중앙에 'PAUSE' 메시지가 표시됩니다.
