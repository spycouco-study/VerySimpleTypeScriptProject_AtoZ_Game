# 게임 프로토타입 사양서

## 1. 게임 개요
비개발자인 사용자가 자연어 입력으로 만들 수 있는 간단한 2D 게임입니다. 플레이어는 특정 키를 사용하여 캐릭터를 조작하고 물풍선을 놓아 게임 내 목표를 달성합니다.

## 2. 핵심 재미
*이 섹션은 사용자 의견을 기다립니다.*

## 3. 플레이어 및 게임 모드
*이 섹션은 사용자 의견을 기다립니다.*

## 4. 조작법
*   **캐릭터 이동:** 키보드의 **방향키** (↑, ↓, ←, →)를 사용하여 캐릭터를 위, 아래, 왼쪽, 오른쪽으로 움직입니다.
*   **물풍선 설치:** **스페이스바**를 눌러 현재 캐릭터 위치에 물풍선을 설치합니다.

## 5. 게임 플레이 메커니즘
*   **물풍선 폭발:** 설치된 물풍선은 **2초** 뒤에 폭발하며, 물줄기가 특정 방향으로 퍼져나갑니다.
